Quick Answer

What is Tencent Hy3 Preview?

Tencent Hy3 Preview is an official preview release in the Hy3 model line. You can verify it through Tencent's official announcement, the Tencent-Hunyuan/Hy3-preview repository, and Tencent-linked model hub pages such as Hugging Face and ModelScope.

For most new work, use the final Tencent Hy3 Model and model ID hy3. Keep Hy3 Preview in your map when you need to understand older examples, compare pricing rows, maintain a legacy hy3-preview integration, or plan a migration.

TokenHub

Hy3 Preview model ID and pricing notes

Tencent Cloud TokenHub documentation and pricing material has referenced hy3-preview as a model ID, with different preview pricing rows depending on input length. Use the official TokenHub API setup and TokenHub pricing pages to verify whether hy3-preview is still available to your account.

  • Do not assume preview pricing equals final hy3 pricing.
  • Check whether your region and account can still call hy3-preview.
  • Log model ID, request ID, token counts, and date of pricing verification for any legacy workflow.

Migration

How to move from Hy3 Preview to final Hy3

  1. Inventory old usage: search code, dashboards, docs, prompts, and invoices for hy3-preview.
  2. Retest prompts: compare output quality, tool calls, structured output, latency, and token cost against final hy3.
  3. Update docs and UI: make sure users see final Tencent Hy3 Model links instead of stale preview-only references.
  4. Keep rollback notes: if preview behavior mattered for a workflow, document what changed and how to detect regressions.

FAQ

Hy3 Preview questions

Is Hy3 Preview official?

Yes. Verify it through Tencent's official article, the Tencent-Hunyuan/Hy3-preview repository, and official model hubs such as Hugging Face and ModelScope.

Should I use Hy3 Preview for a new product?

Usually no. For new work, start with final Tencent Hy3 Model and model ID hy3 when it is available to your account.

What is the Hy3 Preview API model ID?

The preview ID is hy3-preview where Tencent Cloud TokenHub still supports it. Recheck TokenHub documentation before using it.

Why keep a Hy3 Preview page?

It captures search demand from users reading old docs or legacy examples, and it gives them a clear path from preview references to current Tencent Hy3 Model resources.
